# Foxy機器
Foxy機器 添加了各種機器和工具至[Slimefun](https://github.com/Slimefun/Slimefun4).

### 至少需要Java 11.

此為**非官方**版本, 請勿在該作者的問題追蹤內回報!
[原作者連結](https://github.com/GallowsDove/FoxyMachines)
