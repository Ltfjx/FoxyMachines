# Foxy机器
Foxy机器 添加了各种机器和工具至[Slimefun](https://github.com/Slimefun/Slimefun4).
这是GallowsDove 的一个热情专案.

### 至少需要 Java 16.

此为**非官方**版本, 请勿在该作者的问题追踪内回报!
